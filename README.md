## Projeto

### O que o projeto faz?

A ideia do projeto é criar uma página que ajude à encontrar objetos que foram perdidos no campus da Universidade de São Paulo. Para isso, os usuários podem adicionar itens encontrados com a descrição e onde deixaram. Desse modo, caso alguém perca algum objeto, é possível acessar a página e inserir a descrição do item. Assim, o site vai informar uma lista e o usuário poderá encontrar o que perdeu.

### Por que o projeto é útil?

Atualmente, quando um objeto é encontrado na USP os alunos postam em algum grupo de aplicativos de mensagens. Entretanto, nem sempre a pessoa está no grupo ou precisa procurar entre várias mensagens de vários grupos. Desse modo, o projeto proposto permite centralizar a busca em uma única página.

### Como os usuários podem começar a usar seu projeto?

O projeto é apenas um protótipo da página. Entretanto, bastaria acessá-la e adicionar ou buscar os itens de forma intuitiva.

### Onde os usuários podem obter ajuda com seu projeto?

A ideia da página é ser simples e intuitiva.
